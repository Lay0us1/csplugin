## 信息收集 ##
功能:
* usb历史记录信息收集
* 判断是否存在域
* 获取系统盘符
* 获取已安装的软件
* 系统基本信息获取
* 网络信息查询
* 配置查询
* 用户查询
* 用户组查询
* netsh端口转发设置
  * netsh端口转发列表查询
  * netsh端口转发删除
* 常用软件信息收集
  * 向日葵信息收集

## 权限维持 ##
功能：
* cs自带的beacon服务马  
*  winrm后门  
* msdtc DLL劫持  
* WmiPrvSE提权/权限维持
* CacheTask任务计划COM接口劫持 (适用于:windows 10/windows 2012以上)
* RID劫持
* lnk快捷键劫持
* java运行class文件
* dnsadmin账号组进行dns.dll劫持 （适用于域）
* userAccountControl将ID设置为8192作为域控（适用于域）
* 以某个进程的Token运行EXE
* C#添加计划任务
